Sets and Set Operations: Math Grade 7 (MATATAG)

A 
set
 is a well-defined collection of distinct objects, called 
elements
 or 
members
.

Notations

Roster Method:
 List all elements inside braces. Example: A = &#123;1, 2, 3, 4&#125;

Cardinality:
 The number of elements in a set. n(A) = 4

Set Operations

Union (A ∪ B):
 All elements that are in A 
or
 B (or both). Combine everything.

Intersection (A ∩ B):
 Only elements that are in 
both
 A and B. The overlap.

Difference (A − B):
 Elements in A that are 
not
 in B.
